ShadowProfile

Local-first browser profiling transparency instrument.

No Remote Servers
Local Processing
Cross Browser
Portable Exports
Deterministic Releases

What It Does

ShadowProfile helps users understand behavioral telemetry, recommendation activity, cart profiling, tracking pixels, beacon requests, and observable personalization behavior.

How It Works

Everything runs locally inside the browser extension. No account is required. No browsing history is uploaded. No cloud analysis occurs.

Supported Browsers

  • Chrome
  • Firefox
  • Edge
  • Brave
  • Opera

Verification

Releases include:

  • sha256sums.txt
  • release_receipt.json
  • cross-browser deterministic builds
