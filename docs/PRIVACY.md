# ShadowProfile Privacy Policy

ShadowProfile is local-first.

## Data collection

ShadowProfile does not transmit browsing data to external servers.

The extension observes browser-visible signals locally, including request categories, site interactions, cookies, and derived scoring summaries. These observations remain on the user's device unless the user explicitly exports an artifact or later opts into a sync feature.

## Exports

Users may export deterministic JSON session artifacts. These exports are saved locally by the browser.

## Optional future sync

Any future sync feature must be opt-in and summary-first. Raw browsing timelines must not be uploaded by default.

## Third parties

ShadowProfile does not use third-party analytics, advertising SDKs, or remote code execution.
