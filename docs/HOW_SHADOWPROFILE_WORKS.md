# How ShadowProfile Works

1. Passive mode observes baseline site behavior.
2. Deep Inspect mode increases visibility into request categories and behavioral signals.
3. Signals are classified into categories such as cart, telemetry, beacon, tracking pixel, recommendation, and checkout.
4. The popup renders scores, timeline, vendors, endpoints, and explanations.
5. Users can export deterministic JSON artifacts with SHA-256 hashes.
6. Artifacts can be wrapped into verifiable packets using the local packet scripts.
