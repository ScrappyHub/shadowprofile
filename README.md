# ShadowProfile

Local-first browser profiling transparency instrument.

ShadowProfile helps users understand:
- behavioral telemetry
- recommendation activity
- cart and checkout profiling
- tracking pixels
- beacon requests
- personalization signals

Everything runs locally inside the browser.

No remote servers.
No cloud processing.
No account required.

## Features

- Deep Inspect mode
- Passive mode
- Request classification
- Evidence timelines
- Signal explanations
- Portable export artifacts
- Packet verification
- Cross-browser support

## Supported Browsers

- Chrome
- Firefox
- Microsoft Edge
- Brave
- Opera

## Local-First

ShadowProfile performs analysis locally on-device.

The extension does not:
- upload browsing data
- require authentication
- transmit telemetry to ShadowProfile infrastructure
- depend on remote APIs
